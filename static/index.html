Lobster Bridge Indexer

This application indexes bridge events (Deposit and Distribution) from the Sepolia and Holesky networks and provides an API to query them.

API Documentation

Bridge Events Endpoint

GET /eth/bridge/events

Returns a list of bridge events with pagination support.

Query Parameters:

  • event_type - Filter by event type (Deposit or Distribution)
  • network - Filter by network (sepolia or holesky)
  • limit - Number of records to return (default: 10)
  • page - Page number for pagination (default: 1)

Examples:

  • /eth/bridge/events?limit=5 - Get 5 bridge events
  • /eth/bridge/events?event_type=Deposit - Get only Deposit events
  • /eth/bridge/events?network=sepolia - Get only Sepolia network events
