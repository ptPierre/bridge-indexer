This application indexes USDC transfers from the Ethereum blockchain and provides an API to query them.

API Documentation

Transfers Endpoint

GET /eth/transfers

Returns a list of USDC transfers with pagination support.

Query Parameters:

  • limit - Number of records to return (default: 10)
  • page - Page number for pagination (default: 1)

Example:

/eth/transfers?limit=2 - Get 2 transfers
